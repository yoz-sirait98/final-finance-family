# Walkthrough Artifacts

## Feature-Specific Walkthroughs
- Create separate, descriptive Walkthrough documents for distinct features, bug fixes, or design iterations (e.g., `docs/walkthrough_receipt_scanner.md`).
- Do not consolidate unrelated updates or completely erase previous history in a single walkthrough.
